# CCAS.4.3: Software Engineering – Final Project

This project is a web-based Gym Management System developed as part of the CCAS.4.3 Software Engineering course.

The system includes requirement analysis, database design with ERD, and a web-based management interface generated using prompt engineering techniques. The project was tested using Robot Framework, and all test cases passed successfully.

# How to Run the Project

Navigate to the Gym Management System Website folder, download the HTML, CSS, and JavaScript files, then open the HTML file using Live Server.
